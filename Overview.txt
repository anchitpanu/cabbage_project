Mission Overview

Robot is an autonomous agricultural platform designed to:

1. Read an AprilTag (task configuration only)
2. Enter a rectangular planter box
3. Plant exactly 2 seeds with defined spacing
4. Move into cabbage zone
5. Detect cabbages with structured interval logic
6. Stop safely at the end of the planter
7. Report system status via ROS 2

Purpose:
	Detect cabbages
	Detect planter end wall
	Trigger primary stop condition
	

AprilTag Task Encoding

[AB][C][DE]

AB = Seed Gap (cm)
C = Transition Gap
DE = Cabbage Spacing (cm)


Stop Architecture (Layered)

Priority 1 — Emergency Stop
: If obstacle < 0.15 m → STOP immediately

Priority 2 — Vision Stop (Camera 2)
: Use edge of planting box detection
	Vertical wall dominance
	AND
	Horizontal edge disappears
	FOR N frames

Priority 3 — Encoder Safety Stop
: Calculate from all distance
	PLANTER_LENGTH = 1.50
	ENTRY_MARGIN = 0.30
	STOP_DISTANCE = PLANTER_LENGTH + ENTRY_MARGIN
	If distance_inside_planter >= STOP_DISTANCE → STOP
	
	
Entry Strategy

Entry logic:
1. Camera 1 detects long horizontal edge
2. Align parallel
3. Move to correct offset
4. Enter straight
5. Reset encoder
6. Switch to planting mode


Planting Sequence

Reset encoder
Move AB → Plant Seed 1
Move AB → Plant Seed 2
Move C×5 → Start cabbage detection
Move DE repeatedly → Detect cabbage
Monitor stop conditions continuously
